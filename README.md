Project Name: Smart File Organizer

Description:
A Python automation project that automatically organizes files into categorized folders using the os and shutil modules. The program scans a directory, creates folders if they do not exist, and moves files into Images, Documents, Videos, or Others based on their file extensions.

Skills Used:

Python
File Handling
Automation
OS Module
Shutil Module

This is a good beginner GitHub project because it demonstrates practical automation and file management.
